# Multi-Section Quiz Web Application

##  Project Overview
This project is a **multi-section quiz web application** built using **HTML, CSS, and Vanilla JavaScript**.  
It allows users to choose from different quiz sections and answer multiple-choice questions within a timed environment.

The application demonstrates **JavaScript fundamentals**, **DOM manipulation**, basic **state management**, and **user interaction validation**.

---

##  Task Instructions
**Create a quiz application using JavaScript with the following requirements:**

- At least **3 multiple-choice questions**
- Ability to **move to the next question**
- Display the **user’s score at the end**
- Ensure users **cannot proceed without selecting an answer**
- Use:
  - **Basic JavaScript**
  - **DOM Manipulation**

---

##  Implemented Features
The following features were added beyond the basic task requirements:

-  **Multiple Quiz Sections**
  - Technology
  - Psychology
  - General Knowledge
-  **10 unique questions per section** (no repetition)
-  **Timer** that starts when a section is selected
-  **Animated progress bar**
-  **Previous and Next navigation buttons**
-  **Answer validation** (users must select an option before proceeding)
-  **Automatic score calculation**
-  **Clean and user-friendly UI layout**

---

##  Technologies Used
- **HTML5** – Structure of the application  
- **CSS3** – Styling and layout  
- **JavaScript (Vanilla JS)** – Logic, interactivity, and DOM manipulation
